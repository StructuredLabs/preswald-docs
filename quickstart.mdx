## Setup your development

Learn how to set up your environment and start creating lightweight data workflows.

### Install and preview

<AccordionGroup>
  <Accordion icon="terminal" title="Install Preswald SDK">
    Get started by installing the Preswald SDK with pip. Run the following
    command in your terminal: ```pip install preswald``` This will install all
    the core functionalities to help you ingest, transform, and visualize data.
  </Accordion>
  <Accordion icon="play" title="Preview your first pipeline">
    Use the CLI to preview and test your data pipelines locally: 1. Navigate to
    the root of your project where your configuration file is located. 2. Run
    this command to start the development server: ```preswald run``` This allows
    you to preview the pipeline and dashboard in real time.
  </Accordion>
</AccordionGroup>
