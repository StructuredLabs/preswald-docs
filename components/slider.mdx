slider(label: str, min_val: int, max_val: int, step: int, default: int)
Adds a slider to the app for user input.
Parameters:
label (str): Label for the slider.
min_val (int): Minimum value of the slider.
max_val (int): Maximum value of the slider.
step (int): Step size.
default (int): Default value of the slider.
Example:
from preswald import slider

value = slider("Rows to Display", min_val=1, max_val=100, step=10, default=20)
