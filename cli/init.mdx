# `preswald init <project_name>`

Initializes a new project directory with boilerplate files.
Arguments:
project_name (str): Name of the new project.
Example:

preswald init my_dashboard

Generated Files:
hello.py: Starter app script.
config.toml: App configuration file.
secrets.toml: For secure storage of sensitive data.
